O que é um Desenvolvedor Front-End?

O profissional front-end é o encarregado de desenvolver a interface, ou seja, do lado do cliente.

Front-end vem do desenvolvimento de software. Termo utilizado para determinar a parte visual para o cliente.

Trabalho em paralelo com dev e designer.

Antigamente existia apenas o "webmaster" e "webdesigner". Os sites eram praticamente todos estáticos e os editores WYSIWYG
(Ex: FrontPage, Dreamweaver) eram os mais utilizados.

Não existia preocupação com o código e sua manutenção, nem tecnologia adequada para tal fim.

CSS e JS ficavam dentro do mesmo arquivo HTML.

Com as tecnologias ficando cada vez mais populares, foram criados diferentes cargos como por exemplo: webdesigner, programador PHP, entre outros e o desenvolver front-end.

Responsabilidades do
Front-End Developer

O front-end trabalha do lado do cliente (client side) ou seja, diretamente no Browser.

  • Estrutura dos elementos e semântica (HTML)
  • Estilização dos elementos (CSS)
  • Encarregado de criar a interação com o usuário (javascript e CSS)
  • Cross-Browsing
  • Mobile
  • Design Responsivo, Multi-device
  • Otimização da performance (tempo de carregamento e renderização, Progressive Enhancement)
  • Otimização do código para facilitar a indexação (SEO, search engine optimization)
  • Acessibilidade / WAI-ARIA
